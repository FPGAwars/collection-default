## Default Collection

Default collection in *Icestudio 0.2*.

### Blocks
* Bit
  * 0
  * 1
* Config
  * Pull up
  * Pull up inv
  * Tri-state
* Logic
  * Combinational
    * Demux 1:2
    * Demux 1:4
    * Demux 1:8
    * Hex 7 Segment CA
    * Hex 7 Segment CC
    * Mux 2:1
    * Mux 4:1
  * Gate
    * And
    * Nand
    * Nor
    * Not
    * Or
    * Xnor
    * Xnor
  * Sequential
    * Debouncer
    * D flip-flop async
    * D flip-flop
    * T flip-flop async
    * T flip-flop

### Examples
* Basic
  * Led on
  * Switch led
  * Switch and gate
* Counter
  * contadorAsc
  * contadorDes
